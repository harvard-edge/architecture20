# Architecture 2.0: Hello World

Before you read the entire book or dive into the complex tools in the Living Catalog, start here.

Architecture 2.0 proposes a framework called **SARE** (State, Action, Rejection, Evidence). It is the foundational loop required to allow autonomous AI agents to design hardware safely and effectively.

To see how the SARE loop operates in a toy environment, simply run:

```bash
python arch2_loop.py
```

This 50-line script demonstrates exactly where the LLM integrates, where the verification guardrails sit, and how simulation evidence feeds back into the loop. Once you understand this script, you understand Architecture 2.0.
